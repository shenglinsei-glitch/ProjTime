# ProjTime

作業時間予測（PWA）— 项目 / 任务的时间预测与记录工具。

---

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   - `npm install`

2. Start dev server:
   - `npm run dev`

3. Build (outputs to `docs/`):
   - `npm run build`

---

## Deploy (GitHub Pages)

本项目已配置为 build 输出到 `docs/`，可直接使用 GitHub Pages 部署：

- Settings → Pages
- Source: Deploy from a branch
- Branch: `main`
- Folder: `/docs`
